# Sistema de Préstamo de Libros

Sistema web para la gestión de préstamos de libros en una biblioteca. Permite administrar usuarios, registrar libros disponibles y controlar el proceso de préstamo y devolución.

## Descripción

Este proyecto implementa una aplicación web orientada a facilitar la gestión de bibliotecas.  
Permite registrar libros, gestionar usuarios y realizar préstamos de manera organizada.

El sistema está diseñado para apoyar el trabajo del bibliotecario y mejorar el control del inventario de libros.

## Características

- Registro de usuarios
- Registro y gestión de libros
- Sistema de préstamo de libros
- Registro de devoluciones
- Consulta de libros disponibles
- Interfaz web para administración

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- Python
- Django
- MySQL

## Uso

1. Iniciar sesión en el sistema.
2. Registrar libros disponibles en la biblioteca.
3. Registrar usuarios.
4. Realizar préstamos y registrar devoluciones.

## Objetivo del proyecto

Desarrollar una solución web que facilite la administración de préstamos de libros en una biblioteca, permitiendo un mejor control del inventario y del historial de préstamos.
